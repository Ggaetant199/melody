Melody

Docs Exemple

Melody est un framework php qui à pour but de faciliter le démarrage d'un projet php, soit pour créer une nouvelle application en php ou une API complet.

Inspirer de la célèbre librairie express.js pour créer des applications côté serveur avec NODE JS.

include 'vendor/auload.php'; $app = new Melody\main\Main('q'); //middelware comme express.js en nod js $app->get('/', ($req, $res) => { $res->send('route get'); }); $app->post('/', ($req, $res) => { $res->send('route post'); }); $app->put('/', ($req, $res) => { $res->send('route put'); }); $app->delete('/', ($req, $res) => { $res->send('route delete'); }); $app->default(($req, $res) => { $res->statut(404)->send('route par défaut si aucune requête valide '); }); $app->end();
